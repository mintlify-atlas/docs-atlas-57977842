  <h2 className="text-2xl sm:text-3xl font-semibold text-gray-900 dark:text-white mb-3">
    Quick start
  </h2>
  <p className="text-base text-gray-600 dark:text-gray-400 mb-8">
    Get up and running in minutes
  </p>

  <Steps>
    <Step title="Install Geocoder ID">
      Install as a Go library or download the CLI binary.

      <CodeGroup>
      ```bash Go Library
      go get github.com/zsbahtiar/geocoder-id
      ```

      ```bash macOS (Apple Silicon)
      curl -L https://github.com/zsbahtiar/geocoder-id/releases/latest/download/geocoder-darwin-arm64 -o geocoder
      chmod +x geocoder
      ```

      ```bash Linux
      curl -L https://github.com/zsbahtiar/geocoder-id/releases/latest/download/geocoder-linux-amd64 -o geocoder
      chmod +x geocoder
      ```
      </CodeGroup>
    </Step>

    <Step title="Run your first query">
      Query coordinates to get administrative location data. The database will auto-download on first use.

      <Tabs>
        <Tab title="Go">
          ```go
          package main

          import (
              "fmt"
              "log"
              
              geocoder "github.com/zsbahtiar/geocoder-id"
          )

          func main() {
              gc, err := geocoder.NewDefault()
              if err != nil {
                  log.Fatal(err)
              }
              defer gc.Close()

              result := gc.ReverseGeocode(-6.2088, 106.8456)
              
              fmt.Printf("Province: %s\n", result.Province.Name)
              fmt.Printf("Village: %s\n", result.Village.Name)
          }
          ```
        </Tab>
        <Tab title="CLI">
          ```bash
          geocoder geocode --coords="-6.2088 106.8456"
          ```
        </Tab>
      </Tabs>

      <Accordion title="Example output">
        ```
        | province_code | province_name | regency_code | regency_name                      | district_code | district_name | village_code  | village_name  |
        |---------------|---------------|--------------|-----------------------------------|---------------|---------------|---------------|---------------|
        | 31            | DKI Jakarta   | 31.74        | Kota Administrasi Jakarta Selatan | 31.74.02      | Setiabudi     | 31.74.02.1006 | Pasar Manggis |
        ```
      </Accordion>
    </Step>

    <Step title="Explore more features">
      Query multiple coordinates, specify administrative levels, or output as JSON.

      ```bash
      # Multiple coordinates (semicolon separated)
      geocoder geocode --coords="-6.2088 106.8456;-7.2575 112.7521"

      # Specific administrative level
      geocoder geocode --coords="-6.2088 106.8456" --level=district

      # JSON output format
      geocoder geocode --coords="-6.2088 106.8456" --output=json
      ```

      <Info>
        See the [CLI usage guide](/guides/cli-usage) for all available options.
      </Info>
    </Step>
  </Steps>
